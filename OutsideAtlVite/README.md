# OutsideAtl RSC Experience

This package is an integrated Bun workspace for a future React Server Components experience.

It is intentionally separate from the production TanStack Start app in the repository root. Use it for experiments that need Vite RSC primitives without putting starter-demo code in the main app.

## Commands

From the repository root:

```powershell
bun run dev:rsc
bun run build:rsc
bun run preview:rsc
```

Or from this folder:

```powershell
bun run dev
bun run build
bun run preview
```
